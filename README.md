I have already formatted the code and prepared the local git repository for you. 

**To put this website online:**

1.  **Go to GitHub**: Create a new repository named `portfolio` (or `arifulhaquerohan.github.io`).
    *   Do **not** initialize with README, .gitignore, or license.

2.  **Push the Code**:
    Run these commands in your terminal:
    ```bash
    # Replace the URL below with your new repository's URL if different
    git remote add origin https://github.com/arifulhaquerohan/portfolio.git
    
    # Push the code
    git push -u origin main
    ```

3.  **Activate GitHub Pages**:
    *   Go to your repository **Settings** > **Pages**.
    *   Under **Source**, select **Deploy from a branch**.
    *   Select **main** branch and **/(root)** folder.
    *   Click **Save**.

Your site will be live at `https://arifulhaquerohan.github.io/portfolio/`.
